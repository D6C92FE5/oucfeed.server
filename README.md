OUC Feed
====================

中国海洋大学 订阅源

提供中国海洋大学相关的一些网站的 RSS 和 Atom 订阅服务

OUC SRDP #3735 的组成部分


OUC Feed - Server
====================

中国海洋大学订阅源的服务端

储存抓取端推送来的项目，提供 RSS 和 Atom 格式的输出

====================
